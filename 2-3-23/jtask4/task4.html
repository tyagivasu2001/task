Var Keyword

The Scope Of a Variable is a Function Scope
We can redeclare our variables and reassign the values to them
If we use it without delaring it its value will Undefined

Let Keyword

The Scope Of a Let is Block Scope
We can't redeclare our variables but reassign the values to them
We cant use it without initalization ,otherwise it shows refernce error

Const Keyword

The Scope Of a Variable is a Block Scope
We can't redeclare our variables and the value given is constant means not changed
