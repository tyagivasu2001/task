Color property

Css use color values to specify a color typically , these are used to set a color either for the background, border of the element , background color mean color given to the spsce acquire by the elemnet

Way to give color to a Element

  • Hex Code: ##RRGGBB p{color#ff00}
  • Short Hex Code: #RGB p{color:#6A7}
  • RGB%: p{color:rgb(50%,50%.50%)
  • RGB Absolute: p{color:rgb(0,0,255)
  • Keyword:p{color:teal}
