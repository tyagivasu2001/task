Access Modifiers

An Access Modifier is a keyword that we can apply to a member of a class to control ots acess from the outside

There are two type of access modifier

  • public
  • private

Public

By default , all member (properties ,field,methods or function) are public
public access modifier give us a scope to use public member from any where

Private

private member can not acessible from outside the class and it can access only internally with the class
